# AI Resume Analyzer

This project is a Python-based Resume Analyzer built using Streamlit.

## Features
- Upload resume in PDF format
- Extract text from resume
- Detect skills automatically
- Show resume score
- Suggest missing skills

## Technologies Used
- Python
- Streamlit
- PyPDF2

## How to Run

Install libraries:

pip install -r requirements.txt

Run the app:

streamlit run app.py
